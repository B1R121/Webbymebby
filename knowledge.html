Knowledge & Technical Guidance

Dynamic Positioning FMEA

Dynamic Positioning (DP) FMEAs assess the robustness and fault tolerance of vessel DP systems. They are a critical requirement for DP2 and DP3 vessels operating in offshore and marine environments.

DNV Recommended Practice RPD-102

DNV-RP-D102 provides the industry standard methodology for conducting DP FMEAs. It defines documentation, testing philosophy, and acceptance criteria to ensure vessels meet class and client requirements.

View official DNV RPD-102 guidance

Marine & Offshore Risk Analysis

In addition to DP FMEA/FMECA, RMC supports HAZID, HAZOP, and technical risk assessments to improve safety, availability, and regulatory compliance throughout the vessel lifecycle.

Dynamic Positioning Class Matrix

The table below summarises the key differences between DP Classes in accordance with IMO / class guidance.

DP Class Redundancy Single Failure Tolerance Typical Operations
DP Class 1 No redundancy Loss of position may occur after single fault Survey, standby, non-critical operations
DP Class 2 Redundant systems No loss of position after single fault Diving, ROV, construction support
DP Class 3 Physically separated redundancy No loss of position after fire or flooding Drilling, accommodation, critical offshore work
